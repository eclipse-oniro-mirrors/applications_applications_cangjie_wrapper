# 应用仓颉接口

## 简介

应用仓颉接口是在 OpenHarmony 上基于应用子系统能力之上封装的仓颉API。设置应用子系统是 OpenHarmony 系统中预置的系统应用，为用户提供设置系统属性的交互界面，例如设置系统时间，屏幕亮度等系统属性。
